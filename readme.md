## 简介
**用C++11实现的高性能Web服务器** 

## 开发工具
- **操作系统**： centos 7
- **编辑器**： vim + Notepad++
- **编译器**： g++
- **工程构建**： make
- **调试工具**： gdb
- **压测工具**： webbench
## 技术要点
- 并发模型为Reactor。 
- 使用Epoll边沿触发+EPOLLONESHOT，非阻塞IO。
- 为充分利用多核CPU的性能，以多线程的形式实现服务器，并实现线程池避免线程频繁创建销毁造成的系统开销。
- 实现基于小根堆的定时器，用于断开超时连接。
- 实现可以自动增长的缓冲区，作为HTTP连接的输入和输出缓冲区。 
- 实现双异步缓冲日志库，用来调试程序以及记录程序运行状况。 
- 实现mysql连接池，并实现用户注册与登录功能。
